# Indecision App
Front-end web application helps users to make a decision via computer.

## Technologies
* HTML5
* CSS3 preprocessed with SASS 
* JavaScript
* ReactJS
* React Router
* Redux
* Redux thunk
* Firebase
* Jest for testing
* Enzyme for testing
* Webpack
* babel
* NodeJs
